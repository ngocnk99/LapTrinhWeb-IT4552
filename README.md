# LapTrinhWeb-IT4552 -Project


# client
	
	> A Vue.js project

	## Build Setup

	``` bash
	cd client

	# install dependencies
	npm install

	# serve with hot reload at localhost:8080
	npm run dev

	# build for production with minification
	npm run build

	# build for production and view the bundle analyzer report
	npm run build --report

	# run unit tests
	run unit

	# run e2e tests
	npm run e2e

	# run all tests
	npm test
